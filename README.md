> Sales Trend Analysis Using SQL Aggregations :
Objective :
To analyze monthly sales trends by evaluating total revenue and order volume over time using SQL aggregation techniques. This analysis helps identify patterns, seasonality, and overall business performance.

> Dataset Overview :
- Database: MySQL
Table: Online sales data
Time Period Covered: Year 2024

> Key Fields Used:
Date
Transaction ID
Total revenue

> Tools & Technologies :
- Database: MySQL
- IDE: MySQL Workbench

> Core SQL Concepts:
Date conversion and handling
Aggregation functions
Time-based grouping
Sorting and result summarization

> Methodology :
Converted text-based date values into a usable date format
Extracted month and year from order dates
Aggregated total revenue at a monthly level
Calculated monthly order volume using unique transaction counts
Organized results chronologically to observe trends clearly

> Analysis Summary :
Monthly revenue trends were successfully identified
Early months of the year showed higher sales performance
Order volume remained relatively stable across most months
Results are suitable for reporting and dashboard visualization
